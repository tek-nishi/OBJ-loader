
# Wavefront OBJ loader with OpenGL 1.1

![スクリーンショット](screenshot.png "スクリーンショット")

## 概要
OpenGL 1.1 の学習向けにOBJ形式のファイルを読み込むサンプル

## 利用した外部ライブラリ
+ OpenGL 1.1
+ GLFW 3.2.1
+ GLEW
+ stb_image 2.14
